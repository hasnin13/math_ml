# Number Theory
**Number Theory** is the branch of mathematics that focuses on the whole number and their properties, such as divisibility, prime numbers, factors, and remainders.

---

## Number System Overview

### Natural Numbers (ℕ)
- Positive counting numbers: 1, 2, 3, …
- No zero, No negatives.
- Oldest number system used by early humans.
- Cavemen used tally marks.

### Whole Numbers
- natural numbers **plus 0**: 0, 1, 2, 3, …
- Introduced with the concept of "zero."
- Zero used for place-holding (e.g., 10, 1000) 
- Babylonians introduced zero as a placeholder.
